利用manim制作数学动画

发表于 2020-02-21 分类于 数学可视化

前言

3Blue3Brown应该很多朋友都听说过，我认为它是目前最成功的数学视频。他是由斯坦福大学数学系学生Grant Sanderson创建的，他的视频总能给人启发，相信很多朋友看完之后会惊叹：这个视频是怎么做的？

3b1b的动画引擎

我是在知乎上才知道3b1b的动画制作是manim，而且是G.Sanderson自己编写的，非常佩服。很想学一下这个基于python的可视化引擎，但是这个对python有一定的要求，所以我打算把与之相关的介绍文章存放在此，便于以后学习。

知乎用户@李狗嗨和BiliBili用户@Solara为此花了很多精力，威斯康星大学的物理教授Todd Zimmerman写了一份教程，更可贵的是还有与之相关的教程代码放在了他的Github上：

  1. 李狗嗨：3b1b的动画引擎如何配置
  2. 李狗嗨：3b1b动画制作教程 (1)
  3. T.Zimmerman: Getting Started Animating with manim and Python 3.7
  4. Tutorial

目前国内成功的例子

成功的例子有很多，但我要推荐@Solara，我是偶然的机会在BiliBili上看到的，Solara做的相当不错，而且也写了不少教程，有兴趣的可以去他的知乎和b站上看。

结语

希望自己有时间能慢慢摸索出来，加油！
